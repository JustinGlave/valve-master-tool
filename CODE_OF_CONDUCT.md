# Code of Conduct

## Our Standards

This repository is maintained by and for the ATS Inc. team. All contributors and participants are expected to:

- Be respectful and professional in all communications
- Provide constructive feedback focused on the work, not the person
- Keep discussions relevant to the project
- Respect the confidentiality of internal data, workflows, and business processes referenced in this codebase

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks of any kind
- Sharing proprietary ATS data, credentials, or internal information outside authorized channels
- Submitting changes that intentionally introduce security vulnerabilities or data loss risks

## Reporting Issues

If you have a concern about conduct in this project, contact the repository owner directly at justinglave@gmail.com.

## Scope

This code of conduct applies to all activity within this repository — issues, pull requests, commit messages, and any related communications.
